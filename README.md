# NodeLabs

Modern React + TypeScript + Redux Toolkit stack kullanılarak geliştirilmiş bir dashboard web uygulamasıdır.

Bu proje, ölçeklenebilir frontend mimarisi, merkezi state yönetimi ve modüler component yapısını göstermek amacıyla hazırlanmıştır.

##  Features

- React 19 + TypeScript
- Redux Toolkit ile global state yönetimi
- React Router ile sayfa yönlendirme
- Axios ile API entegrasyonu
- TailwindCSS ile modern UI
- Dashboard bileşenleri
- ErrorBoundary ve Loading state yönetimi
- Modüler klasör yapısı

##  Tech Stack

- React
- TypeScript
- Redux Toolkit
- React Router
- Axios
- TailwindCSS
- Vite

##  Installation

npm install  
npm run dev  

##  Purpose

Bu proje, modern frontend geliştirme pratiklerini ve profesyonel proje organizasyonunu sergilemek amacıyla oluşturulmuştur.
